A simple fluid simulation using Unity's particle system. Check the final report paper for more info.

=========================================
How to run
=========================================

1. Create an empty Unity 3D project.
2. Move src/Assets folder to the project root folder. Overwrite SampleScene.unity.
3. Open the project in Unity.
4. There is a Fluid object with SPH (script) compoenent. Adjust its attributes if needed.
5. Adjust fixed timestep in project settings if needed. Suggested value is 0.05.
6. Run. The console prints the actual time in seconds taken by each fixed step.
